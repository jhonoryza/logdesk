# logdesk

A logger desktop app

![image](./logdesk.png)

## Install

```bash
npm install -g yarn
yarn install
```

## Development

```bash
yarn run dev
```

## Build

```bash
# For windows
yarn run build:win

# For macOS
yarn run build:mac

# For Linux
yarn run build:linux

# For ALL
yarn run build
```
