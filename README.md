# logdesk

A logger desktop app

![image](https://github.com/jhonoryza/logdesk/assets/5910636/d9621642-1a2e-4d90-a6c4-bba2d76daf54)

## Install

```bash
npm install -g yarn
yarn install
```

## Development

```bash
yarn run dev
```

## Build

```bash
# For windows
yarn run build:win

# For macOS
yarn run build:mac

# For Linux
yarn run build:linux

# For ALL
yarn run build
```
